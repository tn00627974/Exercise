CSS教學

在這堂課程上面，你了解如何使用CSS設定用在HTML標籤上，以及設定連結顏色。

這章節你認識的CSS設定文字語法

  1. 文字顏色：color
  2. 文字大小：font-size
  3. 文字行距：line-height
  4. 文字字型：font-family
  5. 其他...

下列的emmet語法，哪一項是錯的？

  1. ! = HTML5 環境
  2. link:src = 匯入CSS
  3. p*4 = 產生4個段落
  4. h1 = 產生一個標準的html標籤
  5. lorem = 產生10個單字的亂數文字
